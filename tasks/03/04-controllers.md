### Platesnis paaiskinimas ir reikalavimai

Angliskai, kadangi lietuviskai tai skamba apgailetinai:
Create a controllers directory with your controllers related to already created models. Your controllers should serve all the http methods you gonna use with the callback methods within the class. Use as many HTTP methods as you can, but the required ones are: GET, POST and GET with params.
